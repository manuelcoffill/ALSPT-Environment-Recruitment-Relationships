Folder contains data, code, and plots. To reproduce code, the following steps are recommended:

1. Download full repository by clicking the green "<> code" button and selecting the "Download ZIP" option.
2. Unzip folder into local device
3. Using Rstudio, open the project file "ALSPT environment-recruitment relationships for Github.Rproj"
4. Now that project is open in Rstudio, use the "Files" tab in Rstudio to navigate through repository's folders
5. Following these steps will ensure datasets are appropriately uploaded when reproducing the scripts
